# CODTECH AI Internship – Task 1: Text Summarizer (GUI)

## 🎯 Goal:
Summarize long articles using NLP (T5 transformer model)

## ✅ Features:
- GUI-based interface with default example
- One-click summarization

## ▶ How to Run:

1. Install libraries:
   pip install transformers torch

2. Run script:
   python gui_text_summarizer.py

3. Copy or paste any article. Click "Summarize" to see result
